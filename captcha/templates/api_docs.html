Usage


Get Captcha

Request Parameters
  • Url: http://{{ request.get_host }}{% url 'api_get_captcha' %}
Response
  • captcha - captcha id
  • image - captcha image if exist


Test Captcha

Request Parameters
  • Url: http://{{ request.get_host }}{% url 'api_test_captcha' %}
  • captcha - a captcha id
  • value - the value of the captcha
Response
  • test - True or False.
  • message - Error message if any.

Example


Get a valid captcha id

curl "http://{{ request.get_host }}{% url 'api_get_captcha' %}"

Should give you something like

{
    "captcha": 438,
    "image": "http://localhost:8000/static/captcha/9MQ_Ehbeb30c94-48c3-11e9-86f1-0242ac110002.png"
}

Check the value

curl "http://{{ request.get_host }}{% url 'api_test_captcha' %}?captcha=438&value=123456"

Will give you

{
    "test": true
}

if the value is correct or

{
    "test": false
}

otherwise
